# 🚚 배송 경로 자동 정리 프로그램

엑셀 파일(.xlsx)을 업로드하면 카카오 지도 기반으로 최적 배송 순서를 자동으로 계산해 주는 Windows 프로그램입니다.

---

## 📥 다운로드 및 실행

### 1. 프로그램 다운로드

이 페이지 오른쪽의 **[Releases](../../releases/latest)** 를 클릭합니다.

![Releases 위치](https://img.shields.io/badge/GitHub-Releases-blue)

가장 최신 버전의 **Assets** 항목에서 `배송경로자동정리.exe` 를 클릭해 다운로드합니다.

```
Assets
  └─ 배송경로자동정리.exe   ← 이것을 다운로드
```

> **Python 설치 불필요.** 다운로드한 .exe 파일 하나만 있으면 됩니다.

---

### 2. 실행

다운로드한 `배송경로자동정리.exe` 를 **더블클릭**합니다.

> ⚠️ Windows 보안 경고가 뜨는 경우  
> `추가 정보` → `실행` 을 클릭하면 정상적으로 실행됩니다.  
> (서명되지 않은 개인 개발 프로그램에서 나타나는 일반적인 경고입니다)

---

## 🖥️ 사용 방법

### 준비물

| 항목 | 설명 |
|---|---|
| 카카오 REST API 키 | [카카오 개발자 콘솔](https://developers.kakao.com)에서 발급 |
| 배송 목록 엑셀 파일 | 아래 형식 참고 |

### 엑셀 파일 형식

파일에 아래 열이 반드시 포함되어 있어야 합니다.

| 택배받을 주소 |
|---|
| 광주 북구 문산로 99 ... |

> - `택배받을 주소` 열은 **필수**입니다.
> - `배송순서` 열이 없어도 됩니다. 없으면 자동으로 첫 번째 열에 추가됩니다.

---

### 실행 순서

**① 카카오 API 키 입력**
프로그램 상단에 발급받은 카카오 REST API 키를 입력합니다.

**② 출발 창고/사무실 주소 설정**
`주소 찾기` 버튼을 클릭 → 출발지 주소 검색 → 목록에서 선택합니다.

**③ 엑셀 파일 선택**
`파일 선택` 버튼을 클릭해 배송 목록 엑셀 파일(.xlsx)을 선택합니다.

**④ 실행**
`배송 순서 자동 정리 시작` 버튼을 클릭합니다.

---

## ⏱️ 작업 단계 안내

버튼을 누르면 아래 순서로 자동 진행됩니다.  
배송지 수에 따라 **총 15~25분** 정도 소요됩니다.

| 단계 | 내용 | 소요 시간 |
|---|---|---|
| 1단계 | 엑셀 파일 읽기 | 수초 |
| 2단계 | 각 배송지 위치 확인 | 1~2분 |
| 3단계 | 주소 정확도 검사 | 1~2분 |
| 4단계 | 도로 이동 시간 계산 | 10~15분 |
| 5단계 | 최적 배송 순서 계산 | 최대 3분 |
| 6단계 | 결과 파일 저장 | 수초 |

> **4단계가 가장 오래 걸립니다.** 프로그램이 멈춘 것이 아니니 기다려주세요.

---

## 📄 결과 파일

작업이 완료되면 원본 엑셀 파일과 **같은 폴더**에 결과 파일이 저장됩니다.

```
원본:  배송목록.xlsx
결과:  배송목록_배송순서완성.xlsx   ← 이 파일을 사용하세요
```

결과 파일에는 `배송순서` 열이 1번부터 순서대로 채워져 있습니다.  
원본 파일은 **변경되지 않습니다.**

## ❓ 자주 묻는 질문

**Q. 실행했더니 바이러스 경고가 떴어요.**  
A. Windows Defender가 서명되지 않은 .exe에 경고를 표시하는 것입니다. `추가 정보 → 실행`을 클릭하면 됩니다.

**Q. 카카오 API 키는 어떻게 발급받나요?**  
A. [카카오 개발자 콘솔](https://developers.kakao.com) 접속 → 로그인 → `내 애플리케이션` → `애플리케이션 추가` → 앱 이름 입력 후 생성 → `앱 키` 탭에서 `REST API 키` 복사.

**Q. 4단계에서 너무 오래 걸려요.**  
A. 배송지 60개 기준 약 10~15분이 정상입니다. 중간에 프로그램을 종료해도 다음 실행 시 이어서 진행됩니다.

**Q. 일부 주소 위치를 못 찾았다고 나와요.**  
A. 주소가 너무 상세하거나 오타가 있는 경우입니다. 해당 항목은 배송순서가 비워진 채로 저장됩니다. 엑셀에서 직접 수정해주세요.
